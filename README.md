# j2cl-platform-bom
The J2CL Platform BOM provides a curated set of dependency and plugin versions
for building and maintaining J2CL-based projects.

It centralizes version management for the J2CL compiler, Closure Compiler,
related build plugins, and commonly used libraries in the J2CL ecosystem.

By importing this BOM, projects can avoid version conflicts, ensure compatibility,
and simplify dependency management across multi-module builds.
